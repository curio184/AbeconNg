WebフレームワークとしてAngularを採用し、SPAサイトを実現した。

≪ポイント２≫
WebフレームワークとしてASP.NET Coreを採用した。

どれほど高速であるかはベンチマークを参照
Web Framework Benchmarks

≪ポイント３≫
棲み分けとしては、Angularはフロントとバックエンドを担当し、
ASP.NET CoreはWebAPIサーバーとしてデータの更新業務を担当する。
開発言語は共に相性が良いC#とTypeScript。

≪ポイント４≫
・オリジナルの雰囲気を尊重しつつ、レスポンシブル対応を行った。

≪システム構成≫

ホスティング： Sakura VPS
WebサーバーOS： Ubuntu
Webフレームワーク： ASP.NET Core MVC, Angular
開発言語： C#, TypeScript
Webサーバー： Kestrelで実行されるASP.NETに対して、nginxでリバースプロキシを構成
